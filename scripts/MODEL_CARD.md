# REZ HIVE Model Card

## Models Used
- llama3.2:1b (default)
- llama3.2-vision:11b (vision tasks)
- sovereign-constitutional:latest (guarded responses)

## Capabilities
- Natural language conversation
- Code generation and analysis
- Local file navigation (with consent)
- System monitoring

## Limitations
- No internet access by design
- Model knowledge cutoff applies
- Local hardware constraints

## Ethical Use
- No harmful content generation
- No unauthorized system access
- User consent required for file operations
