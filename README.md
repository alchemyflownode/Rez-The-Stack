## ü¶ä What is Rez Hive?

**Rez Hive** is not a chatbot. It is a **Cognitive Operating System (COS)**. It provides a sovereign layer between your hardware and your intent.

Unlike cloud AI (ChatGPT, Claude), Rez Hive is:
*   **Sovereign:** No API keys. No cloud. Your data never leaves your machine.
*   **Multimodal:** It can **Think** (Logic), **See** (Vision), and **Act** (Execution).
*   **Self-Evolving:** It can rewrite its own rules via the **Mutation Worker**.
*   **Safe:** A **Constitutional Governor** enforces your rules and prevents dangerous actions.
